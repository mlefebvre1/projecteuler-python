# ProjectEuler-Python

#### Solve the first 100 problems from Project Euler in python

https://projecteuler.net

---

### How to use

Install dependencies

 ```shell
pip install -r requirements.txt
```

Run regression tests

```shell
pytest
```

Solve a single problem

```shell
./solve.py -p <problem_number>
```

Solve all problems

```shell
./solve.py
```
